# Pesewa.com - Peer-to-Peer Emergency Micro-Lending Platform

![Pesewa Logo](icons/icon-512x512.png)

Pesewa.com is a revolutionary, trust-based peer-to-peer micro-lending platform that connects communities across Africa for emergency utility loans. Built on human trust via referrals, it enables users to borrow and lend within trusted circles.
